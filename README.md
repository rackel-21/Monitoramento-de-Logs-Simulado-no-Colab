# Monitoramento-de-Logs-Simulado-no-Colab

Tecnologias utilizadas no projeto:
- Python 3
- Pandas
- Regex (re)
- Matplotlib (para visualizações futuras)
- time (monitoramento em tempo real)
- IPython.display (para simular efeito real-time no Colab)

Objetivo do projeto:

**Password Strength Checker** — Verificar a força de senhas e sugerir melhorias.

**Monitoramento de Logs Simulado** — Monitorar logs em tempo real e gerar alertas de eventos suspeitos.
